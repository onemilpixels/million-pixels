# ✅ LAUNCH CHECKLIST — Day-by-Day Execution

## PHASE 0: PRE-LAUNCH (Day 0-5)

### Day 0-2: Foundation
- [ ] Confirm site is live, pixel grid functional, checkout working
- [ ] Install GA4, TikTok Pixel, Reddit Pixel, Google Ads tag
- [ ] Set up server-side GTM for iOS-resilient tracking
- [ ] Verify Stripe webhook → all pixels fire purchase events with correct values
- [ ] Create UTM standards doc (rigid enforcement from day 1)
- [ ] Set up Customer.io / email automation for buyer onboarding
- [ ] Create TikTok Ads Business account + verify
- [ ] Create Google Ads account + complete verification
- [ ] Create Reddit Ads account
- [ ] Apply for verified Twitter/X account (if not done)

### Day 3-5: Creative Inventory
- [ ] Coordinate with Content Agent: lock 15 viral angles
- [ ] Shoot/edit 20 organic videos (founder + UGC mix)
- [ ] Write 30 ad copy variants (Google Search + Reddit promoted)
- [ ] Build 5 landing page variants (different headlines/CTAs)
- [ ] Set up A/B test framework (PostHog, GA4 experiments, or Optimizely)
- [ ] Influencer outreach list: 200 targets across niches

## PHASE 1: ORGANIC BLITZ (Days 1-30)

### Daily Routine
- [ ] Post 3-5 TikToks (Content Agent owns)
- [ ] Post 5-8 Twitter updates + 1 thread
- [ ] Send 20 influencer DMs
- [ ] Engage 30 min on Reddit (target subs, no promo)
- [ ] Reply to all DMs/mentions (community building)
- [ ] Update internal dashboard
- [ ] 30-min competitor scan (rotate sources)

### Weekly Checkpoints
- **Week 1 (Day 7):** Hit 100K total TikTok views, $1K revenue
- **Week 2 (Day 14):** Hit 500K total views, $5K revenue, 3 influencer posts
- **Week 3 (Day 21):** Hit 2M total views, $20K revenue, 1 viral hit (>500K)
- **Week 4 (Day 30):** Hit 5M views, $50K revenue ← **GATE TO PHASE 2**

### Phase 2 Gate Criteria (Day 30)
- [ ] $50K+ cumulative revenue
- [ ] Site conversion rate >2%
- [ ] At least 1 viral organic hit (>500K views)
- [ ] LTV signal: avg customer purchases >1 pixel block
- [ ] Stable tracking infrastructure (no major data gaps)

**IF gate fails:** Hold paid, double down organic for 2 more weeks. Report blockers to Chief Orchestrator.

## RISK REGISTER

| Risk | Likelihood | Impact | Mitigation |
|------|-----------|--------|------------|
| Organic doesn't validate by Day 30 | Medium | High | Hold paid, extend organic 2 weeks, recut content |
| Tracking breaks (iOS, ad blockers) | High | Medium | Server-side tagging from Day 1, post-purchase email confirmation |
| Ad accounts get banned | Medium | High | Multiple backup accounts, BM verification, avoid finance/crypto framing in ad copy |
| Creative fatigue at scale | High | Medium | UGC creator pool, daily creative testing |
| Copycats steal momentum | Medium | Medium | Lean into "original" framing, community moat, faster iteration |
| Landing page conv. rate <2% | Medium | High | Pre-launch CRO sprint, A/B framework live Day 1 |
| Stripe / payment issues at scale | Low | High | Multiple payment processors as backup |
